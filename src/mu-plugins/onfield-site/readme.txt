OnField WordPress Plugin by AIGrowthClub.kr

== Description ==

OnField is a WordPress plugin developed by AIGrowthClub.kr.

== Installation ==

1. Upload the `onfield` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin settings as needed

== Frequently Asked Questions ==

= How do I use OnField? =

After activation, you can access the plugin settings from the WordPress admin panel.
